Glossary

Overview


The Auram magic system is a system that uses combinations of magical glyphs to cast spells.
In this system, all spells are, in some fashion, derived from the 7 base glyphs. Spellcasters using Auram "carve" glyphs into the world (usually into the air, but can be carved into objects), each glyph representing a concept, element, power, etc. Once a spell is put together, the caster then "strikes" the spell, initiating it. This imbues the carved spell into the caster, at which point they can choose to release the spell in several different ways, using gestures, hand signs, spoken words, etc.


Glyphs


Base Glyphs (T0)


All base glyphs are carved using two distinct pieces in order, in the image examples, represented by their color.
Red -> 1st piece
Orange -> 2nd piece


These glyphs can be flipped, as these were made from a left-handed perspective, but right hand dominants would just carve this but like, flipped horizontally.


Base glyphs on their own (not modified by others in making a spell, just cast alone) use large amounts of mana and effort to strike and cast, but are also the strongest by default.

Combining Glyphs into spells


![[glyphcrafting 1.png]]
Red represents the initial/base glyph, specifying the main attributes of the spell.
Orange represents the first modifying glyph, and specifies the modifications to the spell's spread.
Yellow represents the second modifying glyph, and specifies the modifications to the spell's lingering effects.
Green represents the third modifying glyph, and specifies the modifications to the spell's counter.


The effects each glyph has in each position is listed under its image in the [[Auram#Glyphs|Glyphs]] section

Mechanics and Usage


Casting Spells



-- "Shadow wizard money gang, we love casting spells"



To cast a spell, the character specifies what glyph(s) they are using, and then does a check against their [[Character Sheet#Auram|Auram]] if using one glyph, or their [[Character Sheet#Spellcrafting|Spellcrafting]] if using multiple glyphs.


If the character succeeds on their check, they can then specify when and how (with which [[#Gestures (Somatic)|gesture]]) they release the spell.
If the character wishes to continue using the same spell, they can recast it by again rolling the respective check. If the character wishes to release the spell, before or after casting it, they can do so with no check. If they instead wish to release the spell during casting, they must make an [[Character Sheet#Auram|Auram]] check.


Any failed spell check results in one of a few random effects occurring based on the roll of a d20, the list of which is shown below:































Dice Value Effect
Nat 1 Spell backfires
2-5 Spell fails + reduced AP next turn
6-15 Spell fails
16-19 Glyph(s) randomise
Nat 20 No Effect
